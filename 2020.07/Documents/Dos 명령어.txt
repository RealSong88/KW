Dos 커맨드창

도움말
help >>엔터

디렉토리 이동
cd <대상 디렉토리><엔터>

디렉토리 생성
md <디렉토리명>\<디렉토리명>
또는
mkdir <디렉토리명>\<디렉토리명>

드라이브 변경
e: <엔터>
또는
c: <엔터>

. 현재 디렉토리
.. 상위 디렉토리
\ 루트 디렉토리 ====

루트 디렉토리 이동
cd /

모든 폴더 목록을 보여준다
dir /s

절대경로
해당 드라이브의 루트부터 경로를 지정하여 작업을 진행
루트는 윈도우에서는 \이고 리눅스에서는 /입니다.
ex) md \test\a\a2 (같은 드라이브에서 작업하는 경우 드라이브명 생략)
 
상대경로
ex) md e:\test
ex) md e:\test\a

폴더경로의 목록을 보여주는 명령어
ex)tree e:\<폴더명>
